使用html和css可以绘制矩形，三角形和圆形的办法
矩形：

使用border来实现 (图一)border: 10px solid green;

使用box-shadow来实现 (图五)box-shadow: 0 0 0 10px red;

三角形：

使用背景色（图三） 设置下左右三个边框，下边框有颜色，左右透明即可

border-right: 30px solid transparent;

border-bottom: 30px solid green;

border-left: 30px solid transparent;

使用背景色（图四） 设置下左两个边框，下边框有颜色，左边框透明即可

同理可实现其他形状三角形(可根据图二延伸)

圆形：

设置圆角边框为一半即可（图六） border-radius: 50%;
